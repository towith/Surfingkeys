Help

This extension displays the current browser window's tabs in tiles at the top or bottom of the screen.

This is especially useful when viewing a page in full screen (F11) mode, and on large monitors where the usual browser tabs may be small and so difficult to click.

Use the buttons to the left of the tabtiles bar to perform the following actions:

Open a new tab
(Right click) Open a new tab, using the alternate homepage
Go back to the last page
Go forward
Show address / search bar
Go to New tab start page
(Right click) Go to the alternate homepage
Close the address / search bar
(Right click) Hide the tabtiles bar fully (for this page load - updating will show it again)

Move the mouse over the tabtiles bar from side to side to scroll through the tabs. You can also use the mouse wheel for this.

Click on a tab to go to it. Click on a selected tab to allow easily closing it with another click (so this allows double-clicking to close it too - but the tab must be selected first)

You can also right click a tab to pin / unpin it. This switches it to a minimized mode, moving it to the left of the bar.

As well as using the mouse, you can navigate using the keyboard:

  • Use the left and right arrows to select other tabs, and press Enter to go to them
  • Press a key (a - z) to enter an address
  • Alt+\ or F6 (when in full screen) shows the address bar
  • Enter can also close tabs if the current tab is selected - press Enter twice (you need to use the left / right arrows to select it first)
  • Press Esc to cancel the tab selection or close the address bar
  • If the keyboard functions interfere with pages you are viewing, you can disable them in the options above

You can also choose when you want tabtiles to be displayed - either on a normal restored window, a maximized window, or a full screen (F11) window. These can be set independently of each other.

When the address / search bar is shown, enter an address to go to the page, or enter keywords to search. You can press Ctrl+Enter to add the default .com suffix for a simple domain name.

Also supported is integration with Last Tab Back, if it is installed. This extension allows going back to other tabs if there are no pages to go back to in the current tab.

Go back to the last page, or close the tab and go to it's owner tab
(Right click) Go back to the tab's owner, without closing this tab

Note that this extension will not function for special tabs, such as the New tab page or Options / Extension Options tabs (including this page). However, it will allow navigating to / closing these if viewed from a normal tab.
